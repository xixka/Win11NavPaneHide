# Win11 导航栏管理工具

> Description：一键隐藏/显示“图库”“主文件夹”脚本。

一个纯 PowerShell 编写的轻量级脚本，无需安装任何第三方软件，即可轻松管理 Windows 11 文件资源管理器左侧导航栏中的“图库”与“主文件夹”。

## ✨ 功能特性

- **图库管理**：一键隐藏或显示左侧导航栏中的“图库”。
- **主文件夹管理**：支持关闭“主文件夹”，若被系统优化软件误删也可一键恢复。
- **即时生效**：内置“重启资源管理器”选项，更改后无需重启电脑。
- **自动提权**：双击运行自动申请管理员权限。
- **安全无残留**：内嵌 C# 代码突破 `TrustedInstaller` 权限限制，修改注册表后**自动完美还原原始 ACL 权限**，不在系统中残留任何多余权限项。

## 🚀 使用方法

1. 将仓库中的 `Windows 11 图库菜单管理脚本.ps1` 下载到本地。
2. 右键点击该文件，选择 **“使用 PowerShell 运行”**。
3. 脚本会自动弹出 UAC 提示，点击“是”以获取管理员权限。
4. 在弹出的菜单中输入对应数字并回车执行操作：
   - `[1]` 关闭图库 (隐藏)
   - `[2]` 开启图库 (显示)
   - `[3]` 关闭《主文件夹》
   - `[4]` 恢复《主文件夹》
   - `[5]` 重启资源管理器 (使更改立即生效)
   - `[6]` 退出脚本

## ⚙️ 工作原理

Windows 11 的图库和主文件夹属性由系统注册表深处的 `TrustedInstaller` 保护，普通管理员权限无法直接修改。

本脚本通过内嵌 C# 代码调用 `advapi32.dll` 动态启用系统底层令牌（`SeTakeOwnershipPrivilege`），临时接管注册表项所有权并进行修改。修改完成后，严格备份的原始 ACL (访问控制列表) 会被原封不动地写回，恢复 `TrustedInstaller` 所有权，确保系统安全无污染。

## ⚠️ 注意事项

- 本脚本仅适用于 **Windows 11** 系统。
- 保存脚本文件时，请确保编码格式为 `UTF-8 with BOM`，以免中文字符出现乱码导致解析错误。
